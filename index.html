VoiceStudio

Django-платформа для развития навыков выразительного чтения

О проекте

VoiceStudio — веб-платформа для упражнений по литературному чтению с записью голоса, анализом интонации и отслеживанием прогресса.

Технологии

Бэкенд: Django, PostgreSQL
Фронтенд: HTML5, CSS3, JavaScript
Особенность: Web Audio API для записи голоса

Статус

✅ База данных: PostgreSQL настроена
✅ Таблиц: 16 (Django + кастомные)
✅ Пользователей: 3 в системе

Запуск проекта локально

# Клонировать репозиторий git clone https://github.com/Alukarka/VoiceStudio.git cd VoiceStudio # Установить зависимости (если есть requirements.txt) pip install -r requirements.txt # Применить миграции python manage.py migrate # Запустить сервер python manage.py runserver
